Как получить API-ключи
  1. Anthropic Claude: console.anthropic.com → Sign up → Billing (пополни $5) → API Keys → Create Key → копируй ключ sk-ant-...
  2. OpenAI: platform.openai.com → Sign up → Billing → API Keys → Create new secret key → копируй sk-proj-...
  3. Ключи хранятся только в твоём браузере (localStorage), никуда не отправляются кроме как напрямую к API.
