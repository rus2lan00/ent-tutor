# 🎓 ЕНТ Репетитор

Локальный сайт для подготовки к ЕНТ Казахстана по **биологии** и **географии**.

**Никакой установки не нужно.** Просто открой `start.bat` или сам `index.html` — сайт работает прямо в браузере.

## Запуск

1. **Двойной клик на `start.bat`** (или прямо на `index.html`) — сайт откроется в браузере
2. При первом открытии появится окно для ввода **API-ключа от Anthropic** — это нужно сделать один раз

## Получение API-ключа (один раз)

1. Зарегистрируйся на [console.anthropic.com](https://console.anthropic.com/)
2. Раздел **Billing** → пополни баланс хотя бы на $5 (хватит надолго)
3. Раздел **API Keys** → **Create Key** → скопируй ключ
4. Вставь его в окно сайта и нажми «Сохранить»

Ключ хранится только у тебя в браузере (localStorage), никуда не отправляется кроме как напрямую в Anthropic.

## Возможности

- 📚 **78 тем** по программе ЕНТ (биология + география)
- 🤖 **ИИ-конспекты** с ключевыми терминами, фактами и типовыми вопросами ЕНТ
- 🖼️ **Картинки** из Википедии автоматически
- 💬 **Чат с ИИ** в контексте каждой темы
- ⚡ **Кэширование** в браузере — повторное открытие мгновенное

## FAQ

**Сколько стоит?** ~1-2 цента за тему, всего ~$1-2 на весь курс. Чат — 0.1 цента за вопрос. Кэшированное — бесплатно.

**Нужен интернет?** Для генерации новых тем и чата — да. Уже открытые темы хранятся в браузере и работают без интернета.

**Как очистить кэш?** Открой консоль браузера (F12) и выполни `localStorage.clear()`.

**Сменить API-ключ?** Кнопка ⚙️ в правом верхнем углу сайта.

**Добавить свои темы?** Открой `data/biology.js` или `data/geography.js`, допиши новую тему в нужный раздел, обнови страницу.
